A **vector** is a sequence of data elements of the same basic type.

## Changing values in a vector
```
v <- c(5, 6, 7, 8, 9)
v[2] <- 3 # v는 5 3 7 8 9
v <- c(v, 10) # v는 5 3 7 8 9 10
```
```
# append(x, values, after=length(x))
v1 <- 1:3
v1 <- append(v1, 22, after=2) # v1은 1 2 22 3
v1 <- append(v1, c(33, 55), after=3) # v1은 1 2 22 33 55 3
```

## Combining vectors
```
n <- c(1, 3, 5) 
s <- c("a", "b", "c") 
c(n, s) # => "1" "3" "5" "a" "b" "c"
```

## Vector index
```
v <- c(5, 6, 7, 8, 9)
v[3] # => 7
v[2:4]  # => 6 7 8
```
```
# Negative index 
v <- c(5, 6, 7, 8, 9)
v[-3]  # => 5 6 8 9
v[-2:-4]  # => 5 9

v[1:(length(v)-2)]  # => 5 6 7
```

## Numeric index vector
```
s <- c("a", "b", "c", "d", "e", "f", "g") 
s[c(3, 5)]  # => "c" "e"

# Duplicate Indexes
s[c(3, 3, 5)]  # => "c" "c" "e"

# Out-of-Order Indexes
s[c(3, 1, 5)]  # => "c" "a" "e"

# Range Index
s[3:5]  # => "c" "d" "e"
```

## Logical index vector
```
s <- c("a", "b", "c", "d", "e", "f", "g")
LL <- c(FALSE, TRUE, FALSE, TRUE, FALSE, TRUE, TRUE)
s[LL]  # => "b" "d" "f" "g"
s[c(FALSE, TRUE, FALSE, TRUE, FALSE, TRUE, TRUE)]  # => "b" "d" "f" "g"
```

## Named index vector
```
sn <- c("korea", "flower")
names(sn) <- c("first", "second")
sn["first"]  # => "korea"
sn[c("second", "first")]  # => "flower"  "korea"
```

## Vector arithmetics
```
a = c(1, 3, 5, 7) 
b = c(2, 4, 6, 9)
3 * a  # => 3  9 15 21
a + b  # => 3  7 11 16
a - b  # => -1 -1 -1 -2
a * b  # => 2 12 30 63
a / b  # => 0.5000000 0.7500000 0.8333333 0.7777778
```
```
# Recycling rules
u = c(1, 3, 5) 
v = c(1, 2, 3, 4, 5, 6, 7, 8, 9) 
u + v  # => 2  5  8  5  8 11  8 11 14
```
